# Performance Testing Project

This repository contains a performance testing project focused on validating system behavior under load, identifying bottlenecks, and ensuring scalability and stability in real-world scenarios.

## Overview

The goal of this project is to simulate different load conditions and evaluate how the application performs in terms of response time, throughput, and reliability. It is designed as a practical demonstration of performance testing skills, including test design, execution, and reporting.

## Scope

The tests included in this repository cover:

- Load testing (normal expected traffic)
- Stress testing (high load conditions)
- System behavior validation under pressure
- Basic performance metrics analysis

## Technologies Used

- Java
- Maven
- Testing framework (TestNG / JMeter / similar depending on your setup)
- Performance reporting tools

## Project Structure


src/                # Source code for tests
